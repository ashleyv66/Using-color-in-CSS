Using color in CSS (to add colour!)

Some commonly used properties that allow you to set colours

Property name Description Syntax
color Set the foreground colour of an element (Including text) color: orange;
background-color Set the background colour of an element background-color: #12F72A
border-color Set the colour of a border where specified border-color: rgb(78, 236, 159)

Ways to specify a colour in CSS

Type Explanation Example
Keywords Specific color keywords supported in browsers orange
Hexadecimal Hexadecimal number representing a colour #FFA500
RGB value* Specifies levels of Red, Blue and Green (0 - 255) rgb(255, 165, 0)
HSL value* Specifies values of Hue, Saturation and Lightness hsl(39, 100, 50)

*An optional 'alpha' (transparency) value can be added to RGB or HSL by using rgba() or hsla()

E.g.rgba(255, 165, 0, 0.5) where the alpha value is between 0 and 1.
